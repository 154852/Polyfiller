Polyfiller

Create low-poly masterpeices quickly and easily with Polyfiller.

Go to the app

How to use it:

  1. Click to place a point.
  2. Finish a polygon either by right clicking or by using all the sides set (more on that later)
  3. The color is slightly randomised, but the level of randomisation can be changed with the up and down keys. (0 is no randomisation, 100 is loads)
  4. The base color can be set by pressing c
  5. Press q to toggle point locking mode, this is if you click close to the point, it will jump to be at the same place
  6. Press a to toggle point move mode, this is where you can press and drag on a point
  7. Backspace will delete the last full polygon
  8. esc will cancel the current polygon
  9. With shift held, clicking on a polygon will delete it.
  10. Use e to download the image as a png
  11. The size of the canvas can be changed by putting ?size=width,height on the end of the url
  12. All creations are saved using the number displayed when you start as a pointer, they can be loaded again by putting ?load=number to the url
  13. Creations can also be deleted by pressing l
  14. You can get all of the information regarding your canvas with i
  15. To insert a point inside of a polygon, effectively splitting it into 3, hold tab and click inside of a polygon where you want the new point to be
  16. With r pressed, clicking on a polygon will re-color it with the new color settings
  17. You can use the =/+ and - keys to increase or decrease the number of sides per polygon, by default this is 3.
